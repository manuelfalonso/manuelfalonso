# World of Chess

World of Chess is a demo Unity game base in a chess board where you can gain points to summon more pieces

## Technologies

* Unity Engine v2018.2 ([link](https://unity.com/))
* Gimp 2.10 ([link](https://www.gimp.org/))
* GitHub (deploy)
